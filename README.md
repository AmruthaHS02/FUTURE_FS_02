# Client Lead Management System (Mini CRM)

This project is developed as part of the **Future Interns Full Stack Web Development Internship – Task 2**.

## Project Overview

The Client Lead Management System is a simple CRM (Customer Relationship Management) application that helps businesses manage and track potential client leads.

The system allows users to store lead details such as client name, email, source, and status. It helps businesses organize and monitor leads efficiently.

## Features

• Add new client leads  
• Store client information (Name, Email, Source)  
• Track lead status (New / Contacted / Converted)  
• Display leads in a structured table  
• Delete leads when required  
• Data stored using browser Local Storage  

## Technologies Used

HTML5  
CSS3  
JavaScript  

## Skills Demonstrated

• CRUD Operations (Create, Read, Delete)  
• DOM Manipulation  
• Local Storage Data Handling  
• User Interface Design  
• Basic CRM workflow implementation  

## How the System Works

1. The user enters client details in the form.
2. The data is stored in **localStorage**.
3. Leads are displayed dynamically in a table.
4. Users can delete leads when needed.

## Project Pitch

This project is a Client Lead Management System designed to help businesses organize and manage potential customer leads efficiently. The system allows users to store and track client information such as name, email, lead source, and status.

With a simple and interactive interface, businesses can monitor their leads and update their progress through different stages such as new, contacted, and converted. This system demonstrates how web applications can support basic business workflows and improve lead tracking and management.
